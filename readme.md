# Chef Hard Link Testing

## Overview

Tests if chef can update the contents of a file that's been hard linked.

## Directions

* Install Ruby    
* Install Chef with:

    ```bash
    gem install chef
    ```

* Run Chef:

  * OS X:

    ```bash
    ./run.sh
    ```
